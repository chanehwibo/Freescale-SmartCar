历时4个多月，飞思卡尔也告一段落，中间经历双创，创新杯，挑杯以及电赛的赛前准备，等市级各种大大小小的比赛，暂时结束了这忙碌的一段时期

初春的凌晨，实验室还有点小冷，经常一个人通宵调车，逃课也成了家常便饭（吐槽下校园里野猫的叫声太过凄厉，有点}人了哈哈）

看着从车模到框架搭建起来，安装舵机，摄像头，电机，编码器等等

从开环跑1.6m/s,到闭环直上2.2m/s,2.4,到最终赛的2.75，

这一步步都让付出和喜悦都具象化

虽然主要精力在飞思卡尔，但同时多线程比赛，最终没有足够时间做足够的优化，没能再进一步

回想历程，总会懊悔，如果再多点时间，图像处理算法更优化一些，PID算法再准确些，结果会不会不一样，不过也都结束了
